# Домашнее задание к занятию «Репликация и масштабирование. Часть 2»

### Задание 1

Опишите основные преимущества использования масштабирования методами:

- активный master-сервер и пассивный репликационный slave-сервер;
  Преимущества:
### Простота настройки - только 2 сервера, минимум конфигурации

### Отказоустойчивость - если мастер упадет, slave заменит его

### Резервное копирование - можно делать бэкапы со slave без нагрузки на master

### Низкая стоимость - нужен всего 1 дополнительный сервер

### Легкое тестирование - на slave можно тестировать обновления

- master-сервер и несколько slave-серверов;

### Распределение нагрузки чтения - запросы на чтение идут на slaves

### Масштабируемость чтения - можно добавить больше slaves при росте нагрузки

### Параллельная обработка - разные пользователи читают с разных slaves

### Множественные копии данных - данные реплицируются на несколько серверов

### Автоматическое восстановление - если 1 slave упал, остальные работают

### Геораспределение - slaves можно разместить в разных дата-центрах

### Выделенные серверы для отчетов - тяжелые отчеты на отдельном slave

### Безопасное тестирование - тесты на изолированном slave

### Бэкапы без downtime - бэкапим любой slave
